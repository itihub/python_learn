# Peewee ORM 学习

本目录包含使用 Peewee ORM 框架连接和操作数据库的示例代码。

## 学习目标

* 掌握 Peewee ORM 的模型定义。
* 学习如何使用 Peewee 进行 CRUD 操作。
* 理解 Peewee 的查询构建。

## 文件说明

* `models.py`: Peewee 模型定义。
* `crud.py`: Peewee CRUD 操作示例。

## 使用说明

1.  确保已安装 `peewee` 库：

    ```bash
    pip install peewee
    ```

2.  修改 `common/config.py` 文件中的数据库连接信息。
3.  运行示例代码，查看输出结果。
