# ORM 学习概述

本目录包含 Python ORM 框架（SQLAlchemy、Django ORM 和 Peewee）的学习示例。

## 学习目标

* 理解 ORM 的概念和优势。
* 比较不同 ORM 框架的特点和适用场景。
* 学习如何使用 ORM 定义模型、进行 CRUD 操作和处理关系。

## 子目录说明

* `sqlalchemy/`: SQLAlchemy ORM 学习示例。
* `django_orm/`: Django ORM 学习示例。
* `peewee/`: Peewee ORM 学习示例。
